## 🚀 Features

Stop staring at silent red text! **FAhhhh Screamer** actively monitors your coding environment and unleashes the iconic meme audio when things go wrong:

🔥 **Live Code Watching**: If you type bad syntax and leave a red squiggle on the screen, it screams.
💾 **Save & Scream**: Hit `Ctrl+S` on a broken file? It screams before you even try to run it. 
🛑 **Task Crash Detection**: Did your `npm build` or `pytest` task fail? Oh, it screams.
🐛 **Debugger Hooks**: If your Node.js or Python app throws an unhandled exception and hits the debugger, it screams!

## 🎧 Overlapping Chaos
Unlike lesser extensions, FAhhhh Screamer has **no audio locks**. If you rapidly mash save on a broken file or run a failing task 5 times in a row, you will get 5 overlapping, glorious screams simultaneously. 

## ⚙️ Installation

1. Install directly from the VS Code Marketplace.
2. Ensure you have your speakers on!
3. To test it immediately, press `Ctrl+Shift+P` (Command Palette) and type **Test FAhhhh Scream**.

## 🛠️ Requirements & Audio Setup

By default, the extension comes pre-loaded with the authentic `fahhh_KcgAXfs.mp3` audio file. It uses native Windows Script Host (`play.vbs`) to seamlessly play the MP3 in the background without launching any annoying media player UIs.

*Note: The native background player is currently optimized for Windows OS.*
